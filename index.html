Prediction Assignment Writeup - Weight Lifting Exercise

This report is prepared to predict the outcome of weight lifting exercise bsaed on data received from various sensors.

Executive Summary

Random Forest model is fitted to the training data after it is split for cross-validation. The overall estimated error is 0.21%. While performing cross validation it is noticed that there are 6 errors in 4904 test samples in the cross-validation.

Loading and Pre-processing

Below are the basic steps for loading the training and test data.

pmlTraining <- read.csv("pml-training.csv", na.strings = c("NA", ""))
pmlTesting <- read.csv("pml-testing.csv", na.strings = c("NA", ""))
library(caret)

## Loading required package: lattice
## Loading required package: ggplot2

library(randomForest)

## randomForest 4.6-10
## Type rfNews() to see new features/changes/bug fixes.

We remove all columns that have NA since this does not help in the prediction model.

pmlTrainClean <- pmlTraining[, colSums(is.na(pmlTraining)) == 0]

On observing the training data, we can omit the variables related to user, timestamp and window

pmlTrainClean1 <- pmlTrainClean[, -c(1:6)]

Model Selection

Some basic plots of the variables indicates that the relationship between variables seems to be non-linear. Hence a linear model will not be suitable. The best model to apply would be Random Forest since it is very accurate.

qplot(roll_belt, pitch_belt, data = pmlTrainClean1, colour = classe)


qplot(roll_belt, classe, data = pmlTrainClean1)

Cross-Validation

Basic cross-validation is performed on the training set. This will split the training data into further training subset and a test subset. This wil allow us to validate the training model on the test subset.

cvpTrain <- createDataPartition(y = pmlTrainClean1$classe, p = 0.75, list = FALSE)
cpTrain <- pmlTrainClean1[cvpTrain, ]
cpTest <- pmlTrainClean1[-cvpTrain, ]

Model Fit and Prediction

Create the model using RandomForest on the training subset. The model shows an error rate of 0.21%. Applying the model on the test subset indicates that only 6 out of 4904 samples are incorrect.

modPML <- randomForest(classe ~ ., data = cpTrain)
